# my-prose

A simple prose builder library for Java.

## Features

- Build structured prose text programmatically
- Easy to integrate with Maven projects

## Getting Started

### Prerequisites

- Java 17+
- Maven 3.x

### Installation

Clone the repo:

```
git clone https://github.com/Mirakhreis/my-prose.git
cd my-prose
```

Build the project:

```
mvn clean install
```
